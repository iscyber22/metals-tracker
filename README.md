# Metals Tracker

Простой трекер цен на металлы с реальными данными из API.

## Структура

```
metals-tracker/
├── index.html                 # Главная страница
├── netlify.toml              # Конфигурация Netlify
└── netlify/
    └── functions/
        └── metals-api.js     # Прокси функция для API
```

## Установка на Netlify

1. Загрузи этот репозиторий на GitHub
2. Подключи к Netlify (https://app.netlify.com)
3. Netlify автоматически обнаружит `netlify.toml` и развернет функции
4. Готово!

## Особенности

- ✅ Реальные цены из Metals API
- ✅ Кэширование на 15 минут (экономия запросов)
- ✅ 5 металлов: Gold, Silver, Platinum, Palladium, Copper
- ✅ Поддержка 3 валют: USD, EUR, GBP
- ✅ Поиск по названию
- ✅ Мобильный дизайн

## API использование

- **Лимит**: 100 запросов в день (free tier)
- **Обновление**: каждые 15 минут (~96 запросов в день)
- **Кэш**: 15 минут на сервере + браузерный кэш

## Цены обновляются:

- При загрузке страницы
- Каждые 15 минут (автоматически)
- При смене валюты (берет из кэша)
- При поиске (берет из кэша)

Никакие дополнительные запросы к API не делаются за счет сервер-сайд кэширования!
